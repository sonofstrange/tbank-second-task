# T-Библиотека

Консольное приложение для управления личной библиотекой книг.

## Что умеет

- Добавлять книги (название, автор, жанр, год, описание)
- Показывать список всех книг
- Сортировать по названию, автору или году
- Фильтровать по жанру или статусу (прочитана/не прочитана)
- Искать по ключевым словам
- Добавлять в избранное
- Менять статус чтения
- Удалять книги
- Сохранять данные в файл и загружать при запуске

Данные хранятся в файле `library.json` — он создаётся автоматически.

## Меню

```
1. Добавить книгу
2. Просмотр библиотеки
3. Поиск книги
4. Подробности о книге
5. Добавить / убрать из избранного
6. Изменить статус
7. Избранные книги
8. Удалить книгу
0. Выход
```

## Как пользоваться

1. Запустите программу
2. Добавьте несколько книг через пункт 1
3. Просматривайте, сортируйте, ищите — всё через меню
4. Отмечайте прочитанные и добавляйте любимые в избранное
5. При следующем запуске все книги останутся на месте

## main_expanded.py - поиск в интернете

Это версия небольшой доработкой — она умеет искать и импортировать книги из Open Library. Работает только с интернетом, и не гарантирует работы всегда. При старте, если нет `library.json` автоматически импортируется ~100 книг по разным темам.
